# Guess My Card

## Installation

```
npm install
```

## VS Code Extensions

1. Prettier
2. ESLint
3. Jest Runner
4. Live Server
   

## Write Code

1. getRandomInt(max) in random.js
2. Card game logic in app.js

## Test getRandomInt(max)

1. Run Jest from test/random.test.js using Jest Runner
2. Run Jest at command line

```
npm test
```
